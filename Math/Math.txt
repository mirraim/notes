определены в пакете java.lang.Math

Math.abs()
принимает в качестве параметра число и возвращает его абсолютное значение
- т.е. делает передаваемое значение неотрицательным.

int abs = Math.abs(rsl);

Math.pow()
возвращает основание, возведённое в степень
Double x;
Double y;
int n = Math.pow(x, y)

Math.sqrt()
возвращает квадратный корень из аргумента
